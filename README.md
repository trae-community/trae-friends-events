# trae-friends-events

TRAE Friends 是由 TRAE Fellow 发起的城市开发者社区活动，
面向本地开发者与 AI 爱好者，聚焦 AI 编程在真实场景中的实践与交流。

截至目前，TRAE Friends 已在国内累计举办87 场城市活动，
覆盖 50+ 城市，与 8000+ 开发者线下交流。

# What happens here
- Meetup / Workshop / Hackathon / Demoday ……
- 技术分享、实践案例、踩坑记录与动手实践
- 围绕 AI 编程在真实场景中的使用方式展开

# About this repo
trae-friends-events 用于沉淀：
- 城市活动记录与回顾
- 分享内容 / Demo / Hackathon & Workshop 成果

这是一个持续更新、由社区共同维护的仓库。
